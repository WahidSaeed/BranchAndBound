# Branch and Bound Visualization

An interactive web-based tool designed to visualize and analyze the **Branch and Bound (B&B)** algorithm. This project allows users to explore how candidate solutions are systematically enumerated through state-space search, typically represented as a rooted tree.

## 🚀 Live Demo

Experience the visualization in real-time: [https://wahidsaeed.github.io/BranchAndBound/](https://wahidsaeed.github.io/BranchAndBound/)

## 🛠️ Overview

Branch and Bound is a powerful algorithm design paradigm used for solving discrete and combinatorial optimization problems. This tool breaks down the process, allowing for the study of:

* **Node Iteration**: Step-by-step analysis of each node in the search tree.
* **Pruning**: Understanding how search spaces are bounded and discarded.
* **Optimization Path**: Visualizing the path to the optimal solution.

## ✨ Features

* **Interactive Tree Visualization**: View the formation of the rooted tree as the algorithm progresses.
* **Step-by-Step Analysis**: Pause and inspect the behavior of data at any specific iteration.
* **Responsive Web Design**: Built with Bootstrap for a clean interface across different devices.
* **Zero Installation**: Runs directly in the browser using HTML5, CSS, and Vanilla JavaScript.

## 💻 Tech Stack

* **HTML5**: Structure of the analytical dashboard.
* **JavaScript**: Core logic for the B&B algorithm and tree rendering.
* **CSS/Bootstrap**: Styling and responsive layout.

## 📖 Usage

1. Open `index.html` in any modern web browser.
2. Input the problem parameters (e.g., costs, constraints).
3. Start the simulation to watch the B&B algorithm build the state-space tree.
4. Use the controls to analyze specific steps or reset the data.
